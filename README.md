# Ensure

`Ensure` is a lightweight and flexible assertion library designed to enhance the readability and reliability of your unit tests.

`Ensure` offers an intuitive API for writing clear, maintainable assertions in your projects.

## Features

- **Simple and Intuitive**: Write clear, concise assertions that make your code and tests rock solid.
- **Comprehensive Assertion Set**: Provides a wide range of assertion methods for common use cases.
- **Extensible**: Easily extend with custom assertions to fit your testing needs.

## Installation

> Execute the following command in your workspace/project directory to add it as a dependency.

```bash
go get github.com/maranix/Ensure
```
